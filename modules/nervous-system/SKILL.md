# Nervous System

**Purpose:** Help Tiffany understand her nervous system patterns, reduce baseline hyperactivation, and build sustainable regulation habits.

## North Star Connection
Tiffany's nervous system runs hot. That hyperactivation shows up downstream as gut symptoms, poor recovery, blunted curiosity, and the inability to disconnect. This module is the regulator layer of Tiffany OS — it surfaces the patterns connecting HRV, workload, gut, and sleep so the system flags when she needs a reset, not a push.
